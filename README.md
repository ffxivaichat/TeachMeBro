# TeachMeBro

一个用于《最终幻想 XIV》（国服）的 Dalamud 插件，监听聊天消息中的触发关键词，在全屏显示**径向渐变彩色闪烁蒙层**，提醒玩家注意关键信号（如 PvP 撤退、冲锋等）。

## 功能

- **多规则触发** — 支持添加多条规则，每条独立配置地图 ID、关键词、闪烁颜色。第一条命中的规则生效。
- **全屏径向渐变闪烁** — 屏幕边缘浓色、中心透明，约 4Hz 闪烁，持续 3 秒后渐隐。
- **自定义颜色** — 每条规则可独立选择闪烁颜色（颜色选择器）。
- **透明范围调节** — 全局滑块控制蒙层中心透明区域的大小（0% 几乎全屏红 → 100% 仅边缘一圈）。
- **地图过滤** — 每条规则可限定仅在特定地图（Territory ID）下触发，留空则全局生效。
- **批量设为当前地图** — 点击「当前」按钮一键将所有规则的地图 ID 设为当前所在地图。
- **当前地图信息** — 窗口顶部实时显示所在地图 ID、地名、副本名称。

## 命令

| 命令 | 说明 |
|------|------|
| `/teachmebro` | 打开/关闭主窗口 |

## 安装

1. 确保已安装 [Dalamud (CN)](https://github.com/ottercorp/Dalamud)。
2. 在 `/xlsettings` → 实验性 → Dev Plugin Locations 中添加编译输出目录的路径。
3. 在游戏中通过 `/xlplugins` 启用。

> 开发模式下，将 `TeachMeBro\TeachMeBro\bin\Debug`（或 `Release`）目录添加为开发插件路径即可。

## 构建

```bash
git clone <repo-url>
cd TeachMeBro
dotnet restore
dotnet build           # Debug
dotnet build -c Release # 正式版本
```

在安装了 XIVLauncherCN 的 Windows 机器上可直接编译，无需额外配置。
